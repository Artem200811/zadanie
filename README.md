
# Система автоматического подсчёта и напоминания о парах

Программа на Python, которая на основе JSON-расписания подсчитывает количество прошедших и оставшихся пар по дисциплине, вычисляет процент выполнения и сохраняет отчёт.

## Требования
- Python 3.8+
- Стандартные библиотеки (дополнительные зависимости не требуются).

## Структура проекта
- `main.py` — основной исполняемый скрипт.
- `schedule.json` — файл с исходными данными (расписанием).
- `reminder.txt` — файл, в который сохраняется итоговый отчёт (создаётся автоматически).
- `app.log` — файл логирования всех запусков и ошибок (создаётся автоматически).

## Формат входных данных (`schedule.json`)
Файл должен быть валидным JSON и содержать следующие поля:
- `discipline_name` (string): Название дисциплины.
- `group_number` (string): Номер группы.
- `semester_start` (string): Дата начала семестра в формате `YYYY-MM-DD`.
- `semester_end` (string): Дата окончания семестра в формате `YYYY-MM-DD`.
- `schedule` (array): Список объектов с полями:
  - `date` (string): Дата пары в формате `YYYY-MM-DD` или `DD.MM` (год будет взят из `semester_start`).
  - `period` (int): Номер пары (например, 2, 3, 4). Если в день 2 пары, указываются два отдельных объекта.
